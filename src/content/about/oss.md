## Actively maintained

#### [Lokito](https://github.com/mgurov/lokito)

A UI tool (SPA React vite redux) to continuosly pull new logs from loki and provide a convenient "inbox" to capture anything worth attention for manual examination.

#### [GitLabrador](https://gitlab.com/mgurov/gitlabrador)

A UI tool (SPA React vite redux) to continuosly check the status of GitLab Merge Requests and rebase/re-approve/re-trigger failed jobs/merge whenever possible.

## Dormant 

#### [gitlaber](https://gitlab.com/mgurov/gitlaber)

A CLI tool based on Gitlab REST API supporting git-based dynamic infra as a code.

#### [Kibanator](https://github.com/mgurov/kibanator)

A precursor of Lokito for elastic-search/kibana stack. SPA React old school.
